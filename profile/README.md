# Steve Schneider · SUNY Polytechnic Institute

This GitHub organization contains course infrastructure, teaching projects,
research materials, and related academic work developed by Steve Schneider at
SUNY Polytechnic Institute.

## Repositories

- [**stevez**](https://github.com/steve-sunypoly-edu/stevez)
- [**ai-literacy-2026**](https://github.com/steve-sunypoly-edu/ai-literacy-2026)
- [**ai-literacy-2026-raw**](https://github.com/steve-sunypoly-edu/ai-literacy-2026-raw)
- [**rss2zotero**](https://github.com/steve-sunypoly-edu/rss2zotero) — python script with yaml config to harness rss feeds and upload new hits to zotero group via api

## Current focus

The repositories here support courses and projects that examine communication,
information design, artificial intelligence, and the changing relationship
between people and computational systems.

The  [`stevez`](https://github.com/steve-sunypoly-edu/stevez) repository
provides infrastructure and reusable resources intended to support multiple
courses.
